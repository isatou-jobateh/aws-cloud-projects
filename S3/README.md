# AWS S3 Builder Lab

## Overview

In this lab, I explored Amazon Simple Storage Service (Amazon S3), AWS's object storage service. I learned how to create and manage S3 buckets, upload objects, configure permissions, and protect data using versioning.

## Topics Covered

* Created an Amazon S3 bucket
* Uploaded an object to an S3 bucket
* Managed access permissions for buckets and objects
* Created and configured an S3 bucket policy
* Enabled and used bucket versioning

## Skills Gained

* Amazon S3
* Object Storage
* S3 Buckets
* Bucket Policies
* Access Control
* Versioning
* Object Management

## What I Learned

* Amazon S3 stores data as objects inside buckets.
* Buckets must have globally unique names.
* Bucket policies control access to S3 resources.
* Object permissions can be managed separately from bucket permissions.
* Versioning helps protect data by preserving previous versions of objects.
