# AWS EC2 Builder Lab

## Overview

In this lab, I explored Amazon Elastic Compute Cloud (Amazon EC2), AWS's virtual server service. I learned how to launch, monitor, secure, resize, and terminate EC2 instances.

## Topics Covered

* Launched an EC2 web server with termination protection enabled
* Monitored an EC2 instance
* Modified a security group to allow HTTP access
* Resized an EC2 instance to scale resources
* Tested termination protection
* Terminated an EC2 instance

## Skills Gained

* Amazon EC2
* Virtual Servers
* Security Groups
* Instance Monitoring
* Instance Scaling
* Termination Protection
* Web Server Deployment

## What I Learned

* Amazon EC2 provides scalable virtual servers in the cloud.
* Security Groups control inbound and outbound network traffic.
* Instance types can be changed to match workload requirements.
* Termination protection helps prevent accidental deletion of important instances.
* Monitoring EC2 instances helps track performance and resource usage.
